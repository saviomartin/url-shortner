# Url-shortner
A simple cool url shortner built in node js and uses mongodb for the datatbase.

## Live demo
coming soon


## Build
1. Clone the repo
```
gh repo clone saviomartin/zoom-clone
```

2. Installing Dependencies
```
npm install
```

3. Starting server
```
npm start
```
